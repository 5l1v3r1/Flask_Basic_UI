# Flask_Basic_UI
Basic flask user interface using mongodb.

The ui uses login manager and enables you to comfortably login and change your stats.

Prerequisites:

1 - Python

2 - Flask

3 - mongoengine

4 - flask_wtf

5 - flask_mongoengine

6 - flask_login


How to Use:

1 - Clone the repo.

2 - Install the required libraries.

3 - Run the code on your localhost.
